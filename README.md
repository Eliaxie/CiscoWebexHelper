# CiscoWebexHelper

This extension add some event to your CiscoWebex video (Polimi)\

### List of events

* space: play/pause
* ➡️ : forward 30s
* ⬅️ : backward 30s
* ⬆️ : video speed + 0.1
* ⬇️ : video speed - 0.1
* m  : mute/unmute
